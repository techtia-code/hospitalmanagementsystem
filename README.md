# 🏥 MedCore Hospital Management System

A fully responsive hospital management system demo built with vanilla HTML, CSS, and JavaScript. Designed for class presentation purposes.

---

## 🚀 Features

- **Doctor Login System** – Secure login with email and password
- **Personalized Dashboard** – Greets doctor by name (e.g., "Welcome back, Dr. Williams")
- **Patient Records** – Full patient data with diagnoses, ward assignments, vitals, and status
- **Medication Management** – Track medication name, dosage, delivery date, expiry date, and status
- **Doctors Directory** – All staff with specialty, contact, and patient load
- **Daily Schedule** – Per-doctor appointment schedule
- **Search & Filter** – Filter patients and medications by name, status, and more
- **Patient Detail Modal** – Click any patient to view full profile + their medications

## 🖥️ How to Run

1. Clone or download the repository
2. Open `index.html` in any modern web browser
3. Log in with any of the demo accounts above

No build tools, no dependencies, no server required.
